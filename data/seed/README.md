# Демо-данные «Научный клубок»

Синтетический корпус для отладки до получения данных от организаторов хакатона.

## Файлы

| Файл | Содержание |
|------|------------|
| `experiments.json` | 15 экспериментов (источник истины) |
| `experiments.xlsx` | Тот же каталог для теста upload в UI |
| `documents.json` | 9 внутренних отчётов и статей |
| `references/materials.json` | 5 материалов |
| `references/equipment.json` | 9 установок |
| `references/teams.json` | 6 лабораторий |
| `references/topics.json` | 12 тематических тегов |

## Покрытие для демо

- **Материалы:** Ni-Cu сплав, Ni-Cu концентрат, анодный Ni, медный концентрат, вольфрамовый шлам
- **Режимы:** флотация, электролиз, обжиг, термообработка, выщелачивание, автоклав, магнитная сепарация
- **Пробелы:** часть комбинаций material×mode намеренно не исследована (для gap-analysis)
- **Неоднозначность:** «Что делали по электролизу?» → 3 эксперимента (Ni-Cu сплав 220/250°C, анодный Ni) — ассистент просит уточнить материал, затем режим

## Обновление

```bash
# После правки experiments.json:
python scripts/build_demo_xlsx.py
python scripts/seed_data.py
# или POST /api/admin/reload
```

При получении реальных данных — замените файлы в этой папке.
